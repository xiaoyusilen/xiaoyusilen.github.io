Update

When you use golang as basis image, the image you built will be heavy. But we don’t need all of the basis image, so we always choose a light image as far as possible. I recommend you to use alpine as your basis image here. This image is only 3.99MB.

The new image I used alpine built like below.

When you use alpine, you should change command bin/bash to bin/sh, and docker run command should be docker run -it realtime-fetch /bin/sh. If you want to exposure port, please use docker run -p 9000:9000 -it realtime-fetch /bin/sh, the first 9000 is the port of host, the second 9000 is the port of image.
